Livestock Production Trends in Kenya (2021–2023)

Author: Selline Owiti  
Date: 1st July 2025  
Category: Data Science | Research Analysis | Agriculture 

📌 Overview
This project explores livestock production patterns in Kenya between 2021 and 2023, using FAO-style data to generate actionable insights for sustainable and resilient livestock systems.

📊 Key Insights
- Cattle milk production increased by 1.7% from 2021 to 2023.
- Goat milk production rose by 11.85% in 2023 compared to 2022.

📁 Project Structure
- `livestock_analysis.ipynb`: The full Python notebook used for analysis.
- `FOSTAS Data Cleaned_alt.csv`: Cleaned FAO-style dataset.
- `report.md`: Research-style project report.
- `visuals/`: Power BI screenshots of year-by-year dashboards.

🔍 Tools Used
- Python (pandas, matplotlib)
- Jupyter Notebook
- Power BI
- Git & GitHub
